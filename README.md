# مارکت‌لند — فروشگاه چندمنظوره (Next.js)

پیاده‌سازی واقعی و کامل طراحی فروشگاه از Claude Design، با تمرکز روی **راست‌چین (RTL) کامل**، سرعت و سئو.

## زیرساخت

- **Next.js 16** (App Router, React Server Components) + **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **دوزبانه** فارسی (پیش‌فرض، RTL) / انگلیسی (LTR) با مسیر `[locale]`
- فونت **Vazirmatn** از طریق `next/font` (بدون وابستگی شبکه در زمان اجرا)
- تم پویا: رنگ برند، دارک‌مود و گردی گوشه‌ها (CSS variables + localStorage)
- **AI واقعی** (جستجوی زبان طبیعی + چت) با Claude؛ در نبود کلید API، یک موتور تطبیق محلی به‌صورت خودکار جایگزین می‌شود.

## راست‌چین (RTL)

- `dir` و `lang` روی `<html>` در سمت سرور بر اساس locale تنظیم می‌شود (`/fa` → `rtl`، `/en` → `ltr`).
- در سراسر استایل‌ها فقط از خواص منطقی استفاده شده (`inset-inline-start/end`، `text-align:start`، `ms-*/me-*`)؛ هیچ `left/right` فیزیکی شکننده‌ای وجود ندارد.
- آیکن‌های جهت‌دار (فلش بازگشت/ادامه) با کلاس `dir-flip` در حالت RTL آینه می‌شوند.

## اجرا

```bash
npm install
cp .env.example .env.local   # اختیاری: برای فعال‌سازی AI واقعی کلید را وارد کنید
npm run dev                  # توسعه
npm run build && npm start   # نسخه تولید
```

مسیر `/` به‌صورت خودکار به `/fa` هدایت می‌شود.

## صفحات

خانه، فروشگاه (فیلتر/مرتب‌سازی/Grid-List/جستجوی AI)، صفحه محصول (گالری، متغیرها، مشخصات، باکس AI، JSON-LD)، سبد خرید، تسویه، علاقه‌مندی، مقایسه، پنل کاربری (۱۱ بخش)، پنل مدیریت (۹ بخش + استودیوی AI)، بلاگ + مقاله، به‌علاوه `sitemap.xml` و `robots.txt`.
